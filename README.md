# Number Guessing Game
A simple and fun console-based **Number Guessing Game** built with python. Test your luck.

## Features
- choose from 3 difficuily levels: 
 - **Easy:** 0-50, 10 attempts
 - **Medium:** 0-200, 15 attempts
 - **Hard:** 0-500, 20 attempts

- Input validation to ensure valid numbers ( try except)
- Hit system to guide user(very close, close, far away)
- Keeps track of number of attempts
